# Secure API Testing & Authorization Validation

## Project Overview
This project demonstrates API security testing using Postman and a locally created Flask API.

## Tools Used
- Postman
- Python Flask
- Ubuntu Terminal

## Features Tested
- Authentication Testing (Valid, Invalid, No Token)
- Authorization Testing (User ID Access Testing)
- Input Validation Testing
- HTTP Response Analysis
- OWASP API Risk Mapping

## API Setup
The API was created using Flask and runs locally using token-based authentication.
